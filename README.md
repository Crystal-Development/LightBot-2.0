## Installation

Clone:
```sh
$ git clone https://github.com/Crystal-Development/LightBot-2.0.git
```
Then install the package:
```sh
$ npm install
```
Now you can run it:
```sh
$ npm run bot
```
## To manage with ``pm2``
```sh
$ cd /lightbot-2.0/lib
$ pm2 start bot.js --name="NameForTheProcess"
```
